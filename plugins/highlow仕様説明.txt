  概要

  スクラップを賭けてカードのハイ＆ローを当てるギャンブルゲーム。

  ---
  コマンド

  ┌──────────────────────┬──────────────────────────────────────────────────────────┐
  │       コマンド       │                           説明                           │
  ├──────────────────────┼──────────────────────────────────────────────────────────┤
  │ /highlow             │ UIを開く                                                 │
  ├──────────────────────┼──────────────────────────────────────────────────────────┤
  │ /highlow admin reset │ 自分のセッションを強制リセット（highlow.admin 権限必要） │
  └──────────────────────┴──────────────────────────────────────────────────────────┘

  ---
  ゲームフロー

  賭け金を選ぶ → カードが1枚表示される → HIGH or LOW を選択
         ↓                                        ↓
    スクラップ消費                    正解 → 連続数+1、獲得予定スクラップ増加
                                             → 続ける or 換金 を選べる
                                     不正解 → 賭け金没収、ゲーム終了
                                     引き分け（同じ数字）→ 賭け金返還、ゲーム終了

  ---
  賭け金・払い出し

  - 選択肢: 10 / 50 / 100 / 500 / 1000 / MAX（設定値）
  - 払い出し = 最初の賭け金 × ストリーク倍率（複利ではなく固定計算）

  ┌────────────┬────────────────────┐
  │ 連続正解数 │ 倍率（デフォルト） │
  ├────────────┼────────────────────┤
  │ 1回目      │ ×1.9               │
  ├────────────┼────────────────────┤
  │ 2回目      │ ×2.0               │
  ├────────────┼────────────────────┤
  │ 3回目      │ ×2.5               │
  ├────────────┼────────────────────┤
  │ 4回目以降  │ ×3.0               │
  └────────────┴────────────────────┘

  例：100スクラップ賭けて3連続正解 → 250スクラップ獲得予定

  ---
  カード

  - 1〜13のランダムな整数
  - 表示: 1=A、11=J、12=Q、13=K
  - 同じ数字が出たら引き分け（賭け金を返還してゲーム終了）

  ---
  換金

  - 1回以上正解した状態でのみ換金ボタンが出現
  - 換金すると現在の「獲得予定スクラップ」をそのまま受け取りゲーム終了

  ---
  設定（config）

  ┌──────────────────────────────┬──────────────────────┐
  │             項目             │     デフォルト値     │
  ├──────────────────────────────┼──────────────────────┤
  │ 最低賭け金                   │ 10                   │
  ├──────────────────────────────┼──────────────────────┤
  │ 最高賭け金                   │ 5000                 │
  ├──────────────────────────────┼──────────────────────┤
  │ 通常勝利倍率                 │ 1.9                  │
  ├──────────────────────────────┼──────────────────────┤
  │ 連続正解ボーナス倍率テーブル │ [1.9, 2.0, 2.5, 3.0] │
  └──────────────────────────────┴──────────────────────┘
